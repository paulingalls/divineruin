# Bug Tracker — Milestone 9.1 Playtest

## Severity Definitions

| Severity | Definition | Examples |
|----------|-----------|---------|
| **P0 — Blocker** | Session cannot continue | Crash, infinite loop, no audio output, agent disconnects |
| **P1 — Critical** | Major feature broken, workaround possible | Combat doesn't resolve, wrong NPC responds, dice never roll |
| **P2 — Major** | Noticeable problem, doesn't block progress | Wrong ambient audio, HUD shows stale data, latency spike >5s |
| **P3 — Minor** | Cosmetic or minor annoyance | Slight narration inconsistency, UI flicker, minor audio pop |
| **P4 — Polish** | Nice to fix, not blocking playtest | Phrasing could be better, timing slightly off |
